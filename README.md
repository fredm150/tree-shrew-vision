# tree-shrew-vision
University of Virginia | ENGR 1020: Engineering Foundations 2 | Spring 2026

## Project Overview
This project aims to help solve the gap between generating 3D video stimuli for tree shrew vision analysis. We utilize Deep3D for 3D video generation and reconstruction and PyTorch for modeling a custom made convolutional neural network (CNN).

## Tech Stack
* **Language:** Python 3.7+
* **AI/ML:** PyTorch, Deep3D
* **Hardware:**
  * Raspberry Pi 4
  * Arducam 1/4" Multi Camera Adapter Module V2.2
  * Arducam 5MP Sensor Mini Camera Modules
  * 3.5" 480x320 TFT LCD SPI Display Panel
